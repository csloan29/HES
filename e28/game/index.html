Welcome to Evil Hangman 😈

Here's how this works... You guess letters, and I'm going to think of all the words your guesses could possibly be so far. Do you think you can beat me by whittling down my options to only one word? Sounds fun, right?

This concept came from a command line Java application I built while an undergrad. Everything here was build from scratch, I only used the concept and high-level algorithm to start.
